# ProjectFlow - Collaborative Project Management Tool

A professional-grade, real-time collaborative project management application built with **FastAPI** and **React (Vite)**.

## 🚀 Features

- **Real-time Collaboration**: Instant updates across all clients using WebSockets.
- **Kanban Board**: Drag-and-drop tasks between status columns.
- **Task Management**: priorities, due dates, tags, and assignees.
- **Team Communication**: Threaded comments within each task.
- **Smart Notifications**: Alerts for assignments and activity.
- **Progress Tracking**: Dynamic progress calculation based on task completion.
- **Premium Design**: Polished dark-mode interface with smooth animations.

## 🛠️ Tech Stack

- **Backend**: FastAPI (Python), SQLAlchemy, SQLite, JWT Authentication.
- **Frontend**: React 18, Vite, Vanilla CSS (Design System).
- **Communication**: WebSockets for live data syncing.

## 🏁 Getting Started

### 1. Backend Setup
```bash
pip install -r requirements.txt
python projectflow_backend.py
```

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
